## ğŸš€ Overview
**SmartSplit** is a modern, algorithm-driven expense tracking application designed to simplify group finances. Unlike standard split apps, SmartSplit uses a **Graph Simplification Algorithm** (Greedy approach) to minimize the total number of transactions required to settle debts among a group.

Built with **React.js** and **Tailwind CSS**, it features a glassmorphism UI, real-time data visualization, and an AI-powered financial assistant.
